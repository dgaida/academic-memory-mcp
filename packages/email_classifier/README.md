# E-Mail-Klassifikator (email_classifier)

Dieses Package enthält alle Funktionalitäten, die sich um das Training, die Validierung und den Test des E-Mail-Klassifikators drehen.

## Features
- Training von verschiedenen Modellen (TF-IDF, Embeddings, Combined)
- Unterstützung für RandomForest, XGBoost und Transformer
- XAI-Analyse mit SHAP
- Automatisierte E-Mail-Sortierung
- Visualisierung der Datenverteilung

## Installation
Dieses Package kann separat oder als Teil des Hauptprojekts installiert werden.
